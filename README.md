# Edge AI Surveillance

An offline desktop-based surveillance system that performs real-time object detection using Computer Vision and Edge AI without relying on cloud services.

## Features

- Live webcam support
- RTSP camera support (planned)
- Real-time object detection
- Desktop notifications
- Alarm system
- SQLite logging
- Screenshot capture
- FPS & latency monitoring

## Tech Stack

- Python
- OpenCV
- NumPy
- YOLO
- ONNX Runtime
- SQLite
- CustomTkinter
